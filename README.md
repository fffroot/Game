## 📖 О репозитории
`Game` — единое пространство для экспериментов, джемов и готовых игр.  
Каждый подпроект **автономен**: имеет собственные зависимости, систему сборки и документацию. Объединяет их только фокус — интерактивность, геймплей, графика и звук.
